## Reusable templates

### Review template (1200-1500w)
1. Lead (60 words) — what is it, what slot does it fill, who'd care
2. **Specs at a glance** (table: driver / impedance / sensitivity / freq response / price / shell material / cable)
3. **Who it's for** (3 bullet user profiles)
4. **Build, fit, accessories** (200w)
5. **Sound** — split by bass / mids / treble / soundstage / detail (500-700w)
6. **Compared to** (2-3 same-price competitors, 200w)
7. **Verdict** (3 pros / 3 cons / star rating / "buy if" / "skip if")
8. Price + affiliate links (Linsoul, HiFiGo, Amazon)

### Comparison template (1500-1800w)
1. Lead — why these two
2. **Specs side by side** (table)
3. **Sound** — for each axis (bass/mids/treble/stage), declare winner
4. **Fit / build / accessories** — compare
5. **By genre** — which wins for rock, vocals, bass-heavy, etc.
6. **Verdict** — pick A if / pick B if
7. Affiliate links for both

### Buying guide template (2000-2200w)
1. Lead — scope and criteria
2. **TL;DR top pick + runner-up + budget pick**
3. **How we picked** (criteria: tuning, build, accessories, source community sentiment)
4. **#1 Best Overall** (300w)
5. **#2-10** (200w each, mini-reviews)
6. **What we considered but didn't include** (300w, builds authority)
7. **Buying tips** (eartips matter, sources matter, return policy matters)
8. Affiliate links for each

### News template (500-800w)
1. Lead — what was announced, when shipping, MSRP
2. **Specs** (whatever's been officially disclosed)
3. **How it slots in** (vs predecessor, vs competition in same price)
4. **What we'd want to know** (gaps in the announcement)
5. **Pre-order links** (if available)
